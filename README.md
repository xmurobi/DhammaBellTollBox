# DhammaBellTollBox release for upgrade here

About:
DhammaBellTollBox is bell schedule and execute system for Vipassana center.

###  Main features

- **Toll bells automatically by course schedule** - Vipassana center can setup all kinds of courses which taught by S.N. Goenka manually in calender component and the bell tolls will ring automatically days by days. 
- **Setup course automatically by course apply website(Now China Vipassana's center supported only)** - System will sync course website to download all courses and setup by itself automatically.
- **Toll Bell Manually & Pause Bell** - Manually delay the toll or pause it(can resume them later) for disclosure/meal delay. 
- **WiFi & Cable supported** - Server can use cellphone/pad or computer to control the system.
- **Relay supported** - Vipassana center usually use power amplifier to magnify the bell tolls. This system design use relay to control the power amplifier to sound the bell tolls in wide range.
- **Design for industrial** - System design for 7 x 24 x 365 x years continuously running.
- **Web UI** - all device can access the system through web browser.
- **Multiple Languages supported** - Currently Chinese and English supported(Welcome all Dhamma brothers & sisters to join the translation task).
